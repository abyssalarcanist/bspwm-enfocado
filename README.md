# bspwm-enfocado
BSPWM dotfiles

## Required packages (Arch Linux + AUR)
- `bspwm`
- `sxhkd`
- `polybar`
- `picom-ibhagwan-git`
- `alacritty`
- `neovim`
- `vim-plug`
- `rofi`
- `xorg-xset`
- `xsecurelock`
- `feh`
- `dunst`
- `xfce4-power-manager`
- `network-manager-applet`
- `pavucontrol`
### Fonts
- `ttf-iosevka-nerd`
- `nerd-fonts-ibm-plex-mono`
